Welcome to Exploring Spatial

My learning playground for GIS and web development.

Exploring Spatial is a companion site to my blog, Exploring Spatial (see "Recent Posts," on left).  I closed my S-Corporation, Byteworks, in 2010 when I joined Garmin International, but I've resurrected the web site CSS template to host my labs.

Code Repository

All the labs and demos on this site, in fact, the entire site, are on GitHub. Feel free to examine it at GitHub, Inc or pull the repository down to your machine from the HTTPS Clone URL: https://github.com/smitchell/exploringspatial.git

Site Structure

The site is built as a single page web application using Backbone.js. Most of the interesting bits are under /js/views/demos. Going forward, I'll provide links from my blog to the corresponding Backbone views on GitHub.

Labs and Demos

Here are a few of the demos that I've built so far. A couple are still a work in progress:
  • Demo 1 - Shapes and Layers - This example shows layers and shapes. Cloudmade and OSM work great! Google Map, not so much. More work to do here.
  • Demo 2 - ArcGIS Online - This map of earthquakes was created with ArcGIS Online for a Coursera course.
  • Demo 3 - MapBox - This map of earthquakes was generated using TileMill as part of its tutorial.
  • Demo 4 - Garmin Connect Style Map - In this example I'm trying to recreate the map interface used on Garmin Connect using Backbone, Underscore, and Leaflet. I'm just getting started.
