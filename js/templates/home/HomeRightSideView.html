Introduction

This is a companion web site for my blog, Exploring Spatial (see "Recent Posts," on left). My blog is just a personal technology journal where I document what I learn as I explore GIS and mapping software for my job. I've found that documenting what I learn in my blog is a good learning aid. Making my journal public forces me to dig in a little deeper and be more thorough. I closed my S-Corporation, Byteworks, in 2010 when I joined Garmin International, but I resurrected my ByteworksInc.com CSS template to host this labs. I kept the Byteworks logo just because I like it.

The site is built as a single page web site using Backbone.js, so much of the code may be found in Backbone Views under /js/views/demos. Feel free to browse this code, reuse what you see, or offer comments or code refinements.
