Basic Leaflet Example

This example, lifted from Leaflet's Layer Groups and Layers Control tutorial and modified, shows the switching of base map tiles with Leaflet.

I added a Google maps plug-in found in the post Using Google Map tiles with Leaflet. Note the shapes and events don't work with my Google layer. I need to find a different plug-in for Google.

ArcGIS Earthquake Data

This example is the "Hello World" of mapping, the USGS feed of earthquakes for the past 7 days. I used the feed from late July, 2013, to create this earthquake map for one of the lessons in the Geospatial Revolution course.

Mapbox Example

This example is the "Hello World" of mapping, the USGS feed of earthquakes for the past 7 days. I used the feed from late August, 2013, to create this earthquake map for the TileMill Crash Course.

The tutorial has you import the USGS earthquake CSV file, add a legend and change some styles, and then export 6 zoom levels to import into MapBox. Below is a link to the tutorial map tiles in an iFrame.
