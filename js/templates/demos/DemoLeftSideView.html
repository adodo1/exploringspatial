Demo 5 - GeoJSON Feature Collection

This example shows the use of a Leaflet GeoJSON map layer to display a Feature Collection containing my running activities from Garmin Connect. A GeoJSON layer is created using JSON output by a GeoTools FeatureCollection. Each Feature in the collection is an activity summary (no polyline).

It also demonstrates Leaflet's Marker Cluster plugin. The marker cluster map layer is used as a wrapper around the GeoJSON layer.

Finally, this demo shows a click listener on the popup to drill down into the activity. The marker cluster layer is replaced by a Leaflet GeoJSON map layer built from an activity's GeoJSON. The "Return to Search" links swaps the map layers back again.
